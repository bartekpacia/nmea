# nmea

C++ library to parse NMEA 0183 messages.

## Tutorial

This project uses the [Bazel build system](https://bazel.build).

- List all targets:

  ```
  $ bazel query '...'
  ```

- Build all targets:

  ```
  $ bazel  build '...'
  ```

- Build a specific target:

  ```
  $ bazel build //nmea:nmea
  ```

- Run all tests:

  ```
  $ bazel test '...'
  ```
